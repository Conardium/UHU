Vamos a ver variables, alias y a definir y personalizar nuestra forma de trabajar (ficheros de arranque)


*Hay distintos tipos de intérpretes de comandos (bash es el de por defecto)
 -Muestran la misma informacion pero de forma distinta (ej.: ls).
 -Los comandos son los mismos en todos.

· bash ------------> viene por defecto
· ksh (k shell) ---> ejecuta el interprete "ksh"
· csh -------------> ejecuta el interprete "csh"

##Para salir de un intérprete se usa el comando "exit".


############################### VARIABLES (no guarda comandos) ###############################

*Guarda valores(conjunto de caracteres) en esas variables


Sintaxis ==> nomVariable= + "lo que guarda" 

EJEMPLO: A="/home/velez" (puede usarse "" o '')

   *A la hora de usarla, hay que identificar cuando es una variable poniendo delante el '$'

EJEMPLO: cd $A --> al tener el $, lo toma como una variable y no como un "fichero"

*Para visualizar el contenido de una variable se puede hacer con 'echo' o con 'set'(muestra el contenido de todas las variables)
   EJEMPLO: echo $A
            echo $A $B &C
            set

*Para borrar una variable se usa "unset + nomVariable"
   EJEMPLO: unset A	



================ VARIABLES IMPORTANTISIMAS =====================
                                                               =
·HOME -> contiene: "/home/ismael.dapalma060"	               =
·HOSTNAME -> contiene: "polifemo"                              =
·PWD -> contiene la ruta de donde estas (se va actualizando)   =          
·PS1 -> contiene nuestro 'promt' (se puede cambiar)            =
·PS2 -> contiene nuestro promt secundario (se puede cambiar)   =
·USER -> muestra el nombre del usuario 
                                                               =
*La mas importante                                             =
·PATH -> contiene rutas de ficheros ejecutables(verdes) (si no = 
         contiene la ruta donde se encuentre no lo ejecuta)    =
                                                               =
*Para añadir una nueva ruta se usa: PATH=$PATH:LaRuta          =

### ALIAS (guarda comandos) ###
 
*Hace lo mismo que las VARIABLES pero esta vez guarda !comandos¡, como cambiarle el alias a un comando

SINTAXIS: alias + nomVariable= + "comando"

   	EJEMPLO: alias a="ls -la"

*Mostramos todos los alias con: alias
*Borramos un alias con: unalias + nomVariable

Mismo problemas que en las Variables




############################### FICHEROS DE ARRANQUE ###############################

USAREMOS el .profile o .bashrc (se encuentra en nuestro directorio personal) -> hay que usar "ls -a"

· Se usa el fichero '.profile' -> ahí deben guardarse los alias y variables que queramos guardar
   *Se añadade las variables y alias al final del '.profile' con el "joe"

*Para ejecutar el .profile es: ". .profile"

"" -> se ejecuta en el momento y se guarda el resultado
'' -> no se ejecuta en el momento, se guarda en la variable
