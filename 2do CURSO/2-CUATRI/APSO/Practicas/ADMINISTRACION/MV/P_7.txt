###### APRENDER A INSTALAR SOFTWARE DE FORMA GRAFICA Y DESDE EL TERMINAL ####


El software lo descargamos a partir de servidore (repositorios)


###################### INSTALAR PAQUETES GRAFICAMENTE #########################


Buscar en el buscador del ubuntu ---> Software y actualizaciones

Hay 4 tipos de repositorios:

·main -> (siempre tenerlo activo)
·universe -> (hecho por la comunidad)
·restricted -> (para los controladores)
·multiverse -> Quitarlo

·Codigo fuente -> para descargar ficheros fuentes


## EN OTRO SOFTWARE (se almacena otro software de ubuntu) ##
 Activarlo cuando lo necesitemos para un Software en concreto


## ACTUALIZACIONES
dejar las 2 primeras activas y la ultima activarla o desativarla


## AUTENTICACION
Contiene claves(llaves) de firma automatica para poder descargar software dependiendo
del repositorio en el que se encuentre (si se encuentra en la pestaña de autenticacion lo puede descargar)



-------- AÑADIR SERVIDORES (REPOSITORIOS PPA) ----------------

·Nos vamos a 'Otro software' y le damos a añadir.
·Añadimos la linea que queramos y le damos a 'añadir origen'.

·Tambien se puede quitar o modificar una línea PPA.




------- CENTRO DE SOFTWARE ----------

Sirve para instalar paquetes(programas), desisntalar, etc. 




###################### INSTALAR PAQUETES DESDE LA TERMINAL #########################

APTGET -> instala paquetes

APT-CACHE -> se encarga de buscar un paquete sin llegar a instalarlo

APTITUDE -> más potente que el APTGET (hace lo mismo que el APTGET)



¡¡¡¡¡ OJO !!!!!
·Cada vez que se haga un cambio o se añada nuevos repositorios hay que poner:

		sudo apt-get update


·Para realizar comandos de ADMIN, poner delante del comando 'sudo'
	[TE PEDIRA LA CONTRASEÑA DE ADMIN]
